1 RE-re-view and Literate Org-organisating

1.1 Code Blocks

This is about literate coding. About addressing the machine, as well as the human. Text is 'tangled' to address the machine. Addressing the human reader's understanding can be eased by 'weaving' the text.

1.1.1 An ECMAscript Example

An coding style sometimes seen uses an anonymous function to enclose the code. This reduces the amount of name contamination of the global, or pre-existing context.

Object and Functions in ECMAscript are used synonymously. A function object is a basis to use for building and extending objects. 

1:  // A namespace enclosure
2:  var w = (function () {
3:      var w = function () {};
4:      return w;
5:  })()

1.1.2 Object Principles: resuse, and extend

6:  w.extend = function (O, extension) {
7:      for (var p in extension)
8:          O[p] = extension[p];
9:      };
